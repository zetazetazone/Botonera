2. Create `src/services/AudioPlayerService.ts` following RESEARCH.md Pattern 1:
   - Module-level `_player: AudioPlayer | null = null`
   - `initAudioSession()`: calls `setAudioModeAsync({ playsInSilentMode: true, interruptionMode: 'doNotMix' })`
   - `getPlayer()`: lazy-creates player via `createAudioPlayer(null)`, returns it
   - `playSound(uri: string)`: gets player, calls `player.replace({ uri })` then `player.play()`
   - `stopSound()`: gets player, calls `player.pause()` then `player.seekTo(0)` — NO player.stop() (does not exist in expo-audio)
   - `releasePlayer()`: releases and nulls `_player`

3. Create `src/services/FileCacheService.ts` following RESEARCH.md Pattern 3:
   - Use `File`, `Directory`, `Paths` from `expo-file-system`
   - Directories: `sounds`, `thumbnails`, `stickers` under `Paths.document`
   - `ensureDirectories()`: creates dirs if not existing
   - `cacheAudioFile(sourceUri, filename)`: copies to sounds dir, returns dest URI
   - `cacheThumbnail(sourceUri, id, ext='jpg')`: copies to thumbnails dir, returns dest URI
   - `cacheSticker(sourceUri, id, ext='jpg')`: copies to stickers dir, returns dest URI
   - `deleteFile(uri)`: deletes file if exists, warns on error
